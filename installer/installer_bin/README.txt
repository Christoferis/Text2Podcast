#README pls

To use Text2Podcast you have to have the following Programms installed:

- FFMPEG
- Google Tesseract OCR

#FFMPEG
-go to https://ffmpeg.org
-download the right version for your Operating System
-Extract the zip folder and drag it into the "Documents" Folder on your Computer
-go into the "bin" Folder and copy the Path, in the Searchbar search for variables and edit the path variable
-Paste the path into the Path variable
